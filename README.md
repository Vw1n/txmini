# 项目介绍

    浏览器插件demo，具备基本的用户界面交互功能（网页content-script和侧边栏sidepanel）
    
    具体效果是一个简单的todolist小用户界面
    
    没有存储效果，关闭页面后内容会清空

# 使用说明

- 打开Chrome 扩展程序管理界面
- 打开开发者模式
- 点击加载已解压的扩展程序
- 将解压好的文件夹导入
